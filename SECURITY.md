# Security Policy

Agent Handoff Kit is a documentation and template project. It should not
contain secrets, tokens, private logs, or real production data.

## Reporting a Problem

If you find private data or an unsafe example in this repository, open a private
security report if available, or contact the maintainers through the repository
security channel.

## Supported Scope

Security issues in scope:

- accidental private data in templates or examples
- validation gaps that make leaks more likely
- instructions that encourage agents to expose secrets or private logs

Out of scope:

- private forks or local adaptations that publish their own sensitive data
- third-party agent behavior outside this repository
